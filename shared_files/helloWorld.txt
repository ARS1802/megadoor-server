Este computador está conectado a minha rede (roteador, mas o computador está conectado nele via Ethernet) e rodando Ubuntu Server. Quero usar o FastAPI do python para que outros computadores da rede possam fazer requisições HTTPS para este. A relação entre este computador e os demais dispositivos conectados a rede é esta: "Cliente": faz uma requisição com o caminho de um arquivo; "Servidor" (este computador, o que rodará em FastAPI): responde com o arquivo(s) no diretório especificado. Crie este projeto sempre considerando que estou fazendo isso com a finalidade de aprendizado e que minha experiência com a linguagem Python é baixa.
